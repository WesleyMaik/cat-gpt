## Requirements

- node >= v16.16.0
- npm or yarn

## Built with

- ViteJS (React)
- ChakraUI
- Zustand

## Getting Started

1. Clone the project

```
git clone https://github.com/WesleyMaik/cat-gpt
```

2. Install dependencies

```
npm install
```

or

```
yarn install
```

3. Run üî•

```
npm run dev
```

or

```
yarn dev
```
